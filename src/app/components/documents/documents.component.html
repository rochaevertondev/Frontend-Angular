CommonModule no Angular

O CommonModule é um módulo que traz várias diretivas, pipes e funcionalidades básicas utilizadas nos templates do Angular.

Diretivas estruturais

  • *ngIf - Renderiza o elemento se a condição for verdadeira.
  • *ngFor - Itera sobre listas de dados.
  • ngSwitch, ngSwitchCase, ngSwitchDefault - Escolha condicional entre vários elementos.
        <div *ngIf="mostrar">Exibe se 'mostrar' for true</div>
        <li *ngFor="let item of lista">{{ item }}</li>

Diretivas de atributo

  • ngClass - Aplica classes dinamicamente.
  • ngStyle - Aplica estilos inline dinamicamente.
        <div [ngClass]="{{'ativo': isAtivo}}">Conteúdo</div>
        <div [ngStyle]="{{'color': cor}}">Texto colorido</div>

Pipes básicos

  • DatePipe - Formata datas. Ex: {{ '{{ data | date }}' }}
  • UpperCasePipe - Converte texto para maiúsculo. Ex: {{ '{{ nome | uppercase }}' }}
  • LowerCasePipe - Converte texto para minúsculo.
  • CurrencyPipe - Formata valores monetários.
        {{ '{{ hoje | date:"dd/MM/yyyy" }}' }}
        {{ '{{ nome | uppercase }}' }}
        {{ '{{ preco | currency:"BRL" }}' }}
